# Telegraph Clone

The service allows you to publish articles very simply. [Clone Telegra.ph](http://telegra.ph/)
[On Heroku](https://)

# How to use

1. virtualenv -p python3 env
2. source env/bin/activate
3. pip install -r requirements.txt
4. python server.py
